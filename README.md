fork1.c
=======

> A simply ~~wrong~~ correct [fork()](http://goo.gl/zhh3Dx) example in C

---

## Build

Compiles with gcc 4.9.1, without warnings, with the following command exactly:

```bash
gcc -Wall -o fork.out fork1.c
```

## Run

```bash
./fork.out
```
